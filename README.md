# 🎮 Word Antakshari Bot

A Telegram Word Chain Game Bot built with Pyrogram featuring colored buttons, multiplayer support, and real dictionary validation.

## ✨ Features

- 🎨 **Colored Buttons** - Pyrogram ButtonStyle (Red, Green, Blue)
- 👥 **Multiplayer** - 2-20 players per game
- 📚 **Dictionary Validation** - NLTK WordNet + Custom words
- ⏱️ **Turn Timer** - Configurable time limit
- 💯 **Points System** - Score tracking & leaderboard
- ⏭️ **Pass Option** - Skip turn with penalty
- 🚪 **Leave Option** - Exit game anytime
- 🏆 **Winner Detection** - Auto winner announcement
